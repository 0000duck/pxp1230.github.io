数论 Number Theory

在静态类Enclid中，提供了一些有关数论（整数）的方法。

求余数

Euclid.Remainder( 5,  3); // =  2, such that 5 = 1*3 + 2
Euclid.Remainder(-5,  3); // = -2, such that -5 = -1*3 - 2
Euclid.Remainder( 5, -3); // =  2, such that 5 = -1*-3 + 2
Euclid.Remainder(-5, -3); // = -2, such that -5 = 1*-3 - 2

求模

应用场景比如整队报数 0 1 2 0 1 2 0 1 2

Euclid.Modulus( 5,  3); // =  2, congruent modulo 3 by 5 - 1*3
Euclid.Modulus(-5,  3); // =  1, congruent modulo 3 by -5 + 2*3
Euclid.Modulus( 5, -3); // = -1, congruent modulo -3 by 5 + 2*-3
Euclid.Modulus(-5, -3); // = -2, congruent modulo -3 by -5 - 1*-3
