# 前情提要 #
- <div>
	n：分子数密度
	</div>
- <div>
	$\bar Z$：平均碰撞频率，单位时间内一个分子与其他分子发生碰撞的平均次数
	</div>
- <div>
	d：分子的有效直径
	</div>
- <div>
	${\bar \lambda }$：平均自由程，连续两次碰撞之间自由通过的路程
	</div>
- <div>
	k：理想气体状态方程中的常量
	</div>
# 平均碰撞频率公式 #
<div>
$\bar Z = \sqrt 2 \pi {d^2}\bar vn$
</div>
# 平均自由程公式 #
<div>
$\bar \lambda  = \frac{{\bar v}}{{\bar Z}} = \frac{1}{{\sqrt 2 \pi {d^2}n}} = \frac{{kT}}{{\sqrt 2 \pi {d^2}p}}$
</div>
